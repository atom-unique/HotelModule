## HOTEL

Электронный администратор гостиницы

### Технологический стек

- Spring boot starter Web
- Apache Tomcat
- Spring data Jpa
- MapStruct
- PostgreSQL
- Lombok
- Liquibase
- Open API
- Log4j2
- TestNG
- Mockito

### Правила сборки

1) docker-compose up -d
2) Убедиться, что контейнеры успешно запущены
3) mvn clean compile
4) run application
